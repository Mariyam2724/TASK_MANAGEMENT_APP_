# Final Reflection

## Project Summary

This project helped me understand how a full-stack MERN application is built from start to finish. I developed a Task Management Application that allows users to create, edit, delete, search, filter, and manage their daily tasks. I also deployed the frontend and backend so the application can be accessed online.


## Challenges Faced

Some of the challenges I faced during development were:

- Connecting the React frontend with the Express backend.
- Fixing API and routing errors.
- Understanding CRUD operations.
- Deploying the backend on Render and the frontend on Vercel.
- Resolving deployment issues caused by incorrect API URLs and environment variables.

These challenges helped me improve my debugging and problem-solving skills.


## What I Learned

Through this project, I learned:

- React component development.
- State management using React Hooks.
- REST API development using Express.
- MongoDB Atlas integration with Mongoose.
- CRUD operations.
- API integration using Axios.
- Git and GitHub workflow.
- Deploying a MERN application using Render and Vercel.


## AI Usage Experience

ChatGPT was used as a learning assistant throughout the project. It helped me understand concepts, debug errors, and improve different parts of the application. However, I implemented, tested, and deployed the project myself.


## Future Improvements

Given more time, I would like to add:

- User authentication.
- Task categories and priorities.
- Responsive design improvements.
- Dark mode.
- Drag-and-drop task management.
- Task reminders and notifications.


## Conclusion

This project gave me practical experience in developing and deploying a MERN Stack application. It also improved my confidence in working with full-stack technologies and strengthened my understanding of the complete development process.
