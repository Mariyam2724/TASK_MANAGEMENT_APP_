# AI Usage Documentation

## AI Tool Used

- ChatGPT

## Purpose

ChatGPT was used as a learning and development assistant during the project. It helped me understand concepts, resolve development issues, and improve different parts of the application.

## AI Assisted In

- Understanding the MERN project structure.
- Resolving frontend and backend errors.
- Suggesting improvements for the user interface.
- Providing deployment guidance.
- Reviewing project documentation.

## Example Prompts

- How can I connect React with an Express backend?
- Why is my API returning a 404 error?
- How can I deploy a MERN application?
- How can I improve the project UI?

## Manual Work

The following work was completed by me:

- Setting up the project structure.
- Creating React components.
- Developing backend APIs.
- Connecting MongoDB Atlas.
- Integrating frontend and backend.
- Testing the application.
- Managing Git commits and GitHub repository.
- Deploying the application.

## Note

AI suggestions were used only for guidance. The project implementation, debugging, testing, and final decisions were completed manually.
