# Task Management App

## Overview

This project is a Task Management Application built using the MERN Stack. It helps users manage their daily tasks by allowing them to create, edit, update, and delete tasks. Users can also search tasks, filter them by status, and view task statistics on the dashboard.

## Features

- Add a new task
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Filter tasks by status
- Set a due date
- Dashboard showing task statistics
- Toast notifications for user actions

## Technology Stack

### Frontend
- React.js
- Vite
- CSS
- Axios
- React Router
- React Toastify

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose


## Installation

Clone the repository:

git clone https://github.com/Mariyam2724/TASK_MANAGEMENT_APP_.git


Install frontend dependencies:

cd client
npm install


Install backend dependencies:

cd ../server
npm install

## Environment Variables

Create a `.env` file inside the **server** folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string

## Run the Application

Start the backend:

cd server
npm run dev


Start the frontend:

cd client
npm run dev


The frontend will run on **http://localhost:5173**.

The backend will run on **http://localhost:5000**.
